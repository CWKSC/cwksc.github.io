# Problem 1.4 Sum of Random Variables

Let $x$ and $y$ be random variables (possibly dependent), show the following property:
$$ \mathbb{E}[x + y] = \mathbb{E}[x] + \mathbb{E}[y]. \tag{1.8} $$
Furthermore, if $x$ and $y$ are statistically independent ($x \perp y$), show that
$$ \text{var}(x + y) = \text{var}(x) + \text{var}(y). \tag{1.9} $$
However, in general this is not the case when $x$ and $y$ are dependent.
